【win10实用小技巧】谷歌浏览器书签固定到开始菜单屏幕（磁贴）当中

Posted on 2019-07-13
  1. 将目标书签拖动到桌面，在桌面上形成该书签的快捷方式。


  2. 将桌面的书签快捷方式复制到C盘如下目标文件夹下（C:\ProgramData\Microsoft\Windows\Start Menu\Programs）


  3. 打开开始菜单，就能看到书签了，右键固定到开始屏幕。


  4. 结果完美。
