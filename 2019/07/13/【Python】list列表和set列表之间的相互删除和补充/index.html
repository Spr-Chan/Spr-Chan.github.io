  很奇怪’Adam’删掉了，’Lisa’并没有被删掉，于是增加set和list的元素之后再做尝试，输出结果为[‘Lisa’, ‘cyc’, ‘Paul’]。

  于是发现规律：当list列表中的一个元素被删掉，那么紧接着这个被删掉的元素的后面的元素(‘Lisa’和’cyc’)就不会被删掉，不管’Lisa’和’cyc’有没有跟set里的元素重复。所以看上去就好像’Lisa’和’cyc’被跳过了。
