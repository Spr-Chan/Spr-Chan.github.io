【C++】一图搞懂“从栈底往栈顶压、从栈顶往栈底压”两种情况

发表于 2019-07-11 更新于 2019-07-16 分类于 C++ 评论数：

  设栈的存储空间为S(1:m)，表示栈内存储m个元素。



  通常情况，栈都是从栈底往栈顶压，初始状态栈顶指针top=0（栈空）。当top=m+1，栈满；当top=n（n<=m），栈内有n个元素。

  当从栈顶往栈底压时，初始状态栈顶指针top=m+1（栈空）。当top=0，栈满；当top=n（n<=m+1），栈内有(m-top+1)个元素。
